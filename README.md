# Claude Code Hub 文档站

[Claude Code Hub](https://github.com/ding113/claude-code-hub) 的官方在线文档。

## 技术栈

- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS 4
- **文档**: Markdoc
- **主题**: Claude 品牌色彩系统

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
